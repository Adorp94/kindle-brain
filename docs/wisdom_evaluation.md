# Kindle Brain — Wisdom Evaluation Framework

## Methodology: Inversion Thinking

Instead of asking "how do we make it wise?", we ask: **"What questions would expose the system as NOT wise?"**

A truly wise system should:
1. Pull from diverse, high-signal books (not just the most generic matches)
2. Connect ideas across intellectual traditions (stoicism + startups + psychology)
3. Surface unexpected but relevant connections
4. Cite the heaviest-hitter books when they're relevant (not obscure ones with shallow insights)
5. Synthesize a meta-pattern, not just list quotes

- Steve Jobs (follow your curiosity, intersection of technology and liberal arts)

- Shoe Dog (Phil Knight just started, trusted his gut)

- Feynman (curiosity-driven life, pleasure of finding things out)
- The Element (Ken Robinson — finding your element)

**Ideal search strategy:**
1. "encontrar propósito y pasión en la vida" → broad
2. "Steve Jobs curiosidad intersección artes y tecnología" → targeted
3. "Naval Ravikant specific knowledge build" → targeted
4. "Phil Knight Nike starting journey" → biography
5. "resistance creative work turning pro Pressfield" → creative angle

**Anti-pattern (what a dumb system does):** Returns only Sharma and Robbins self-help books.

## Evaluation Criteria

For each test question, score:

| Criterion | Score | Description |
|-----------|-------|-------------|
| **Book diversity** | 0-5 | How many different books cited? (5 = 6+ books) |
| **Heavy hitters** | 0-5 | Did it cite the most relevant iconic books? |
| **Unexpected connections** | 0-5 | Did it connect books from different genres? |
| **Meta-pattern** | 0-5 | Did it find the deeper insight across sources? |
| **Author voice** | 0-5 | Did it use direct quotes effectively? |
| **Total** | /25 | |

**Target: 20+/25 per question = truly wise system**

## How to Run Evaluation

```bash
# Test each question via curl and count unique books in response
curl -s -X POST http://localhost:8765/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "¿Cómo encuentro mi propósito o mi gran idea?"}' | \
  python -m json.tool | grep book_title | sort -u | wc -l
```

Or via the SwiftUI app — expand sources and verify book diversity.
